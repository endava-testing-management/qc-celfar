Celfar: Celsius to Fahrenheit Converter

User Story: Convert Celsius to Fahrenheit

  • As a user I'll be able to convert temperatures from Celsius to Fahrenheit

AC: acceptance criteria

  • The title page should be "CelFar"
  • The app Title should be “CelFar”
  • A sub title: Celsius to Fahrenheit temperature converter (the words Celsius and Fahrenheit has a wikipedia link asociated)
  • As a user Im able to insert a temperature in celsius to be converted to Fahrenheit
    • Numeric Field:
      • Max lenght should be 6 digits
      • Decimal separator should be ","
      • Should be empty by default
    • Button: "convert!"
    • Result: should be "Calculation Pending.."
    • As a user when I enter a valid temperature in Celsius and click convert
      • The result is displayed as (x*9/5)+32 (where x is the entered temperature)
    • As a user when I enter a temperature with less than 6 digits (without the decimal separator and the sign) and less than -273
      • The following message is displayed: "The value entered is below the absolute 0” "
    • As a user when I left the temperature empty or a enter a not numeric value
      • The following message is displayed: "The value entered is not a number: (remember that the decimal should be expresed with '.' and not with ',') "
    • As a user when I enter a temperature with more than 6 digits (without the decimal separator and the sign)
      • The following message is displayed: "The value entered is too long” "
  • As a user when I update the temperature
    • The result is changed to "Calculation Pending.."
